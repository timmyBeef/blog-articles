最近要來玩火紅的 Flutter ! 先來看看如何安裝

windows 10 installation

到這下載 zip 檔案
https://flutter.dev/docs/get-started/install/windows

解壓縮到 C:\src\flutter\bin

設定環境變數

設定好後可以測試幾個指令

flutter --version

flutter doctor

安裝 android studio

https://developer.android.com/studio

安裝好後打開

安裝 flutter plugin

裝好後 就可以看到有 flutter 的選項

建立一個新的 flutter 專案

在 adroid studio 新建一個 flutter 專案

一直下一步

create simulator

如果 HAMX 裝不起來, 一定要裝, 等等才能跑模擬

因為我有裝 docker 所以 Hyper-V 被啟動了, 手動關掉
