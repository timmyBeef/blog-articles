本篇透過 google search console 的一些設定來完成 網站 SEO

1. 首先要到 Google Search Console 註冊你的網站

我是選擇右邊的方式，輸入你的網站網址後

會要你下載這個檔案，放到你的網站目錄下

放入後像是這樣

2. 產生 sitemap.xml

HAXO 有一個很棒的 plugin 可以幫你產生 sitemap.xml

安裝後，設定 _config.yml

如此一來在 hexo generate 的時候就會幫你產生 sitemap.xml

接著跑 hexo deploy 到網站上檢查 sitemap.xml 是不是存在了

以我的部落格舉例來說，輸入 https://timmybeef.github.io/sitemap.xml 就會看到以下畫面

裡面包含了其他的 sitemap.xml，我這個例子有三個 sitemap分別是

post 的頁面就是 https://timmybeef.github.io/post-sitemap.xml

tag 的頁面就是 https://timmybeef.github.io/tag-sitemap.xml

page 的頁面就是 https://timmybeef.github.io/page-sitemap.xml

3. 新增 sitemap 到 google search console

在 google sitemap 官網有提到，可以透過 robots.txt 或 直接把 sitemap 提交到 search console， 我是選擇直接提交到 search console

https://support.google.com/webmasters/answer/183668?hl=zh-Hant&ref_topic=4581190

所以我就到了 sitemap 這個畫面，把剛剛那三個 sitemap 網址新增進去

如此就大功告成! 之後就可以透過 google search console 來查看網站瀏覽的狀況~~
