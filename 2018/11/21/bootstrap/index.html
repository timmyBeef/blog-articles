因為又要為之前的客戶做教育訓練…所以整理了一些之前專案用的技術，必學的部分，比較進階的就不教了XD

Bootstrap 3

RWD - 響應式網頁設計(Responsive Web Design)

優點:

  • 適應不同螢幕尺寸
  • 開發維護成本低
  • 分享網址好用
  • GOOGLE 愛用
  • 不用到 APP 商店下載

缺點:

  • 須注意載入時間(因為圖片需要手機版一張, 網頁版一張), 可利用工具檢測
  • 瀏覽器版本過低不支援(IE6, IE7)
  • 瀏覽動線問題(版面問題)
  • 功能取捨(版面問題)
